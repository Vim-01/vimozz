# Vimozz

Vimozz is a Twitch Music Bot with YouTube integration and anti-censorship bypass capabilities. It allows streamers to play music requested by their viewers seamlessly, while bypassing network restrictions using an integrated Xray proxy tunnel.

## Features
- **Twitch Integration**: Automatically listen for channel point redemptions and chat commands.
- **YouTube Playback**: Play YouTube videos directly via an embedded player.
- **Anti-Censorship Bypass**: Integrated VLESS/Reality Xray proxy support to ensure YouTube and other services remain accessible even in restricted networks.
- **Queue Management**: Sync and manage track queues in real-time.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Set up your `.env` file with your Twitch API credentials.
4. Run `npm run dev` for development, or use `npm run build` to build the app.
